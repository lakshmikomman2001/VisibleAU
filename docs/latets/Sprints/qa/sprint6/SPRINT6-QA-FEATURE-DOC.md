# VisibleAU Sprint 6 — QA Feature Document (Claude Code)

**Version:** 1.0
**Sprint:** 6 — Action Center (research-backed recommendations from audit scores)
**Purpose:** Feature-specific E2E QA tests for Sprint 6. Each feature has its own `.bat`
(Windows) and `.sh` (Unix/macOS) script that seeds real test data, starts the Next.js dev
server, runs Playwright tests, then hard-deletes all seeded rows on exit — pass or fail.

**Sprint 6 critical invariants baked into every test:**

- `action_items` has RLS **ENABLED** (org-scoped) — `recommendation_research` has RLS **DISABLED**.
- Unique index `(auditId, recommendationKey)` on `action_items` — idempotent generation.
- `PATCH /api/action-items/[id]/status` endpoint is at `/[id]/status/` NOT `/[id]/` (DM3 fix).
- `expectedImpactScore` field is lowercase `'high'|'medium'|'low'` — NOT `'High'|'priority'` (DH2 fix).
- `dismissedReason` is **required** when `status='dismissed'` (Zod `.refine()`).
- `status='in_progress'` clears `doneAt=null` and `dismissedAt=null`.
- `evidenceRefs` on `action_items` are **immutable** — PATCH never updates them (DH5).
- Inngest event name: `'audit/complete'` (slash, NOT `'audit.complete'`) (DA3 fix).
- `generateRecommendations` must be registered in `serve()` (DB5 fix).
- `Object.groupBy` must NOT be used — use `reduce` polyfill (DL1 fix, Node 20 compat).
- `recommendation_research` rows are seeded; `action_items` are generated by Inngest.
- Sprint 8 scope (NOT in Sprint 6): effort, timeToImpact, engines fields, Dashboard KPI card, re-audit trigger.
- Free tier: titles visible, action text **blurred**, upgrade CTA. Starter+: full content.
- Cross-org on all action-item routes: **404** (RLS-enforced), NOT 401.
- 11 valid `recommendationKey` values: `wikipedia-article`, `au-local-citations`, `stale-content`, `faq-content`, `expert-quotes`, `cited-statistics`, `comparison-article`, `reddit-absence`, `medium-presence`, `linkedin-presence`, `press-mentions`.
- 12 BLOCKED_KEYS that must never appear in generated recommendations.
- Confidence labels: `confirmed` (green), `likely` (amber), `hypothesis` (gray) — 3 values only.

**Threshold that triggers ALL 11 action types (used in Inngest + audit job tests):**

- `scoreFrequency='10.00'`, `scorePosition='10.00'`, `scoreAccuracy='10.00'`,
  `scoreSentimentNumeric='10.00'`, `scoreContextNumeric='10.00'`, `scoreComposite='10.00'`

**Prerequisites:** Sprints 1–5 accepted. `pnpm seed` ran (recommendation_research populated).
RLS enabled on `action_items`. `generateRecommendations` registered in `serve()`.

### `tests/qa/sprint6/shared/cleanup.ts`

```typescript
import { db }       from './db';
import * as schema  from '../../../db/schema';
import { eq, like, inArray } from 'drizzle-orm';

// Deletes action items seeded with [S6-QA] prefix in title.
// Does NOT touch real recommendation_research rows (global seed data).
export async function cleanupActionItems(orgId: string): Promise<void> {
  if (!orgId) return;
  await db.delete(schema.actionItems).where(eq(schema.actionItems.organizationId, orgId));
}

// Full org cleanup: action_items → citations → audits → brands → users → organizations
export async function cleanupOrg(orgId: string): Promise<void> {
  if (!orgId) return;
  // 1. action_items (new in Sprint 6 — must be deleted before audits due to FK)
  await db.delete(schema.actionItems).where(eq(schema.actionItems.organizationId, orgId));
  // 2. citations (Sprint 3 FK to audits)
  const auditRows = await db.select({ id: schema.audits.id })
    .from(schema.audits).where(eq(schema.audits.organizationId, orgId));
  if (auditRows.length > 0)
    await db.delete(schema.citations)
      .where(inArray(schema.citations.auditId, auditRows.map(a => a.id)));
  // 3. audits
  await db.delete(schema.audits).where(eq(schema.audits.organizationId, orgId));
  // 4. brands → users → organizations
  await db.delete(schema.brands).where(eq(schema.brands.organizationId, orgId));
  await db.delete(schema.users).where(eq(schema.users.organizationId, orgId));
  await db.delete(schema.organizations).where(eq(schema.organizations.id, orgId));
}
```
